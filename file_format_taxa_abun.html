FishTaco file formats¶

Format of taxa abundance file¶

A tab-delimited file where each row is a taxon and each column is a sample (with headers):

Taxa Sample1 Sample2 Sample3
Taxon1 0.2 0.5 0.1
Taxon2 0.3 0.1 0.7
Taxon3 0.3 0.2 0.1
Taxon4 0.2 0.2 0.1
